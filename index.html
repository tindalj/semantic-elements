Semantic Elements enhance:

  • accessibility
  • searchability
  • internationalization
  • interoperability

Understanding Semantic Elements is important to web development.

The goal of Semantic Elements is to clearly describe their meaning to both the browser and developer.

Semantic Elements

Many web sites contain HTML code to indicate navigation, header, and footer.

Some examples include:
  • header
  • footer
  • nav
  • section

They work as labels for the tools being used on that section of the website. They are an efficient way to keep code organized.
